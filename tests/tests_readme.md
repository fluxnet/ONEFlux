## Test instructions

/Currently only applicable to ustar_cp regression and unit tests/ 

### Test dependencies

(all available on pypi)

- Python >=3.10
- matlabengine >= 24.1.2
- Pytest >= 8.3.2
- pytest-cov >=5.0.0
- pytest-progress >=1.3.0

### How to run

1. Navigate to local top level `ONEFlux` dir in terminal
2. enter `pytest` in terminal
